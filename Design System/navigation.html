Navigation

Navigation should follow the navbar above, with logotype on the left linking to the home page, and links on the right using flexbox. The nav bar should also collapse into a hamburger menu at smaller screen sizes. To use this navbar, copy the HTML, CSS from the codepen linked below, changing the logotype and the target, label, and number of links as needed, and include the Javscript from the codepen in all pages using the nav bar.
