Navigation

Navigation should follow the navbar above, with logotype on the left linking to the home page, and links on the right using flexbox. The nav bar should also collapse into a hamburger menu at smaller screen sizes. To use this navbar, copy the HTML, CSS from the codepen linked below, changing the logotype and the target, label, and number of links as needed, and include the Javscript from the codepen in all pages using the nav bar.

See the Pen Navigation by Tyson Foster (@Foster-Design-System) on CodePen.

The colours used for the navbar are as follows:

Navigation

Background:

#191919 rgba(25, 25, 25, 1)

Text:

Default: #00aeef rgba(0, 174, 239, 1)

Hover: #ffffff rgba(255, 255, 255, 1)

Visited: #73daff rgba(115, 218, 255, 1)

The hamburger icon should be coloured white #ffffff when the hamburger menu is open, and when the menu is closed it should be white if the main background is dark, or black #191919 when the main background is light
