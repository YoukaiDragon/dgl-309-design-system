Foster Design System

Cards

Cards should be used to display a group of similar items, each one containing an image, description or summary, and, if appropriate, a link

Card borders should be 4px solid rgba(25, 25, 25, 1) with a 5% border radius, and card images should have a matching border radius on top (border-radius: 5% 5% 0 0)

The background colour is flexible, but should be low opacity (0.4) when the main background is a light colour, and high opacity (0.8) when the main background is a dark colour

To add cards to a page, copy the HTML and CSS from the codepen linked below, changing the content and removing the links if they are not needed

To add additional cards to the example layout, create additional copies of a .card div inside of the main .cards div
