Cards

Cards should be used to display a group of similar items, each one containing an image, description or summary, and, if appropriate, a link

Cards have a 5% border radius, and card images have border-radius: 5% 5% 0 0; to match the card

The background colour is flexible, but should be low opacity (0.4) when the main background is a light colour, and high opacity (0.8) when the main background is a dark colour
