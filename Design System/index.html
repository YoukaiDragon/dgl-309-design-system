Images

Aspect Ratios

  • 16:9 Banner / "Hero" Images, Section Backgrounds
  • 4:3 Horizontal Card Image
  • 3:4 Vertical Images
  • 1:1 Favicons, Social Media Icons

Image Styling

Images should have a 1px solid black border

Border radius should be set to either 0, 5%, or 10%. Rounded corners are preferred unless it is desireable to keep the image edges flush with the container / other images.

Profile / Avatar images are preferred to have a 50% border radius when not used as a background

Spacing

Use "box-sizing: border-box" for consistent spacing of content

Paddings and margins should be multiples of 4px where possible

Content should be spaced using a 12-column "grid" layout using flexbox, with content spanning multiple columns based on width of the content and the window

Buttons: 0px, 4px, 8px, or 12px vertical padding, 8px horizontal padding.

Images: 12px margin between images and text
