Foster Design System

Images

Aspect Ratios

  • 16:9 Banner / "Hero" Images, Section Backgrounds
  • 4:3 Horizontal Card Image
  • 3:4 Vertical Images
  • 1:1 Favicons, Social Media Icons

Image Styling

Images should have a 1px solid border, rgba(25, 25, 25, 1) against light backgrounds or rgba(255, 255, 255, 1) against dark backgrounds, or they should have no border.

Border radius should be set to 0 for most images, and should be set to 5% 5% 0 0 (5% on the top, 0 on the bottom) for images used in cards

Profile / Avatar images are preferred to have a 50% border radius when not used as a background
