Forms

Labels should be "display: block" for text and textarea fields, and "display: inline for other fields.

Label should be placed after radio and checkbox inputs, and before other inputs

Radio buttons example

Checkbox example



CSS rules for forms:

form { border: 1px solid black; display: inline-block; padding: 8px; }

label { margin: 4px 0; }

for text / textarea labels:

.textInput label { display: block; }

input { margin-bottom: 4px; }

.submit { color: white; background-color: rgba(4, 128, 64, 0.8); border: 1px solid black; border-radius: 10%; padding: 4px 8px; margin: 12px 0 4px; }

.submit:hover { background-color: rgba(4, 128, 64, 1); }

Lists

list-style-position should be set to "outside"

When using a border around a list, place the list in a div and use the following styles along with the border on the div:

"display: inline-block;" and "padding-right: 24px;"
